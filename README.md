# <こどもたちの居場所>

## サイト概要
### サイトテーマ
*こどもたちが各々の気持ちや思い・悩みなどを投稿できたり、投稿に対してコメントができる
*大人（親）がこどもたちに相談したり意見を聞くことができるコミュニティサイト

### テーマを選んだ理由
共働きをしている知り合いのお子さんが不登校になってしまい、コミュニケーションが取れていないという話を聞いて、
自分自身が母子家庭で育ったこともあり、こどもたちが思いや悩みを吐露できる場所があったらいいなと思い、このテーマにしました。

### ターゲットユーザ
*悩みなどを吐露できる場所がないこどもたち
*こどもたちの気持ちを知りたい、意見を聞きたい親たち

### 主な利用シーン
*【こども】周りの目を気にせずに自分の気持ちや悩みを吐露したい時
*【親】こどもたちとの向き合い方がわからない時
